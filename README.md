# ✨ GenStory – AI Story Generation Platform

GenStory is a full-stack web application that generates interactive and dynamic stories based on user inputs. 
It allows users to create personalized narratives using AI-driven generation workflows with a modern and responsive UI.

---

## 🚀 Live Demo
👉 https://gen-story-theta.vercel.app/

---

## 📌 Features

- 🎭 Generate interactive stories based on user prompts
- 🧠 AI-powered dynamic storytelling workflow
- ⚡ Real-time story rendering
- 🎨 Clean and responsive user interface
- 🔗 Seamless frontend–backend integration
- 📱 Fully responsive design for all devices

---

## 🛠️ Tech Stack

**Frontend:**
- React.js
- HTML5
- CSS3
- JavaScript

**Backend:**
- FastAPI
- Python

**Others:**
- REST APIs
- Prompt-based AI logic

---

## 🏗️ System Architecture

User Input → React Frontend → FastAPI Backend → Story Generation Logic → Response Rendered in UI

---

## ⚙️ How It Works

1. User enters a story prompt or selects preferences  
2. Frontend sends request to FastAPI backend  
3. Backend processes input and generates structured story content  
4. Response is sent back and rendered dynamically in UI
